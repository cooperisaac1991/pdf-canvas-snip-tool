## What PDF Canvas Does

PDF Canvas is designed as a web-based workspace for reading and reviewing PDFs directly in the browser. It keeps the focus on practical document work, letting you open multiple files, annotate passages, and navigate pages without needing a heavier desktop application.

Since everything is handled on the client side, it works well as a self-contained browser experience. The project also supports install-as-app behavior, so it can be added to a device and used more like a lightweight app while still remaining web-native.
